# Base Images Catalog

This is a list of the official base images provided by the Dev Containers project, hosted on the Microsoft Container Registry (`mcr.microsoft.com`).

## Universal Image
A large, general-purpose image with many common tools installed.
*   `mcr.microsoft.com/devcontainers/universal`
*   Variants: `linux`, `noble`, `5-noble`, `5.1-noble`, `5.1-linux`

## Base Operating Systems
*   **Debian**: `mcr.microsoft.com/devcontainers/base:debian` (Variants: `trixie`, `bookworm`, `bullseye`)
*   **Ubuntu**: `mcr.microsoft.com/devcontainers/base:ubuntu` (Variants: `noble`, `jammy`)
*   **Alpine**: `mcr.microsoft.com/devcontainers/base:alpine` (Variants: `3.23`, `3.22`, `3.21`)

## Language Specific Images
*   **C/C++**: `mcr.microsoft.com/devcontainers/cpp`
*   **Python**: `mcr.microsoft.com/devcontainers/python`
*   **Node.js / JavaScript**: `mcr.microsoft.com/devcontainers/javascript-node` (Variants: `24-bookworm`, `22-bookworm`, `20-bookworm`)
*   **Node.js / TypeScript**: `mcr.microsoft.com/devcontainers/typescript-node`
*   **Java**: `mcr.microsoft.com/devcontainers/java` (Variants: `25-bookworm`, `21-bookworm`, `17-bookworm`, `11-bookworm`)
*   **Java 8**: `mcr.microsoft.com/devcontainers/java:8`
*   **Go**: `mcr.microsoft.com/devcontainers/go` (Variants: `1.26-bookworm`, `1.25-bookworm`, `1-bookworm`)
*   **Rust**: `mcr.microsoft.com/devcontainers/rust`
*   **.NET**: `mcr.microsoft.com/devcontainers/dotnet` (Variants: `10.0-noble`, `9.0-bookworm`, `8.0-jammy`)
*   **PHP**: `mcr.microsoft.com/devcontainers/php` (Variants: `8.5-bookworm`, `8.4-bookworm`, `8.2-bookworm`)
*   **Ruby**: `mcr.microsoft.com/devcontainers/ruby` (Variants: `3.4-bookworm`, `3.3-bookworm`, `3.2-bookworm`)
