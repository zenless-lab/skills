# Tables Reference

ReStructuredText provides two syntax variants for delineating table cells: Grid Tables and Simple Tables. Tables are also generated by the `csv-table` and `list-table` directives.

## Grid Tables
Grid tables provide a complete table representation via grid-like "ASCII art". Grid tables allow arbitrary cell contents (body elements), and both row and column spans.

- **Markers:** `-`, `=`, `|`, `+`.
- **Header:** Use `=` for optional header rows.

```rst
+------------------------+------------+----------+----------+
| Header row, column 1   | Header 2   | Header 3 | Header 4 |
+========================+============+==========+==========+
| body row 1, column 1   | column 2   | column 3 | column 4 |
+------------------------+------------+----------+----------+
| body row 2             | Cells may span columns.          |
+------------------------+------------+---------------------+
| body row 3             | Cells may  | - Table cells       |
+------------------------+ span rows. | - contain           |
| body row 4             |            | - body elements.    |
+------------------------+------------+---------------------+
```

## Simple Tables
Simple tables provide a compact and easy to type but limited row-oriented table representation. Cell contents are typically single paragraphs.

- **Borders:** Made up of `=` and `-` characters.
- **Top/Bottom:** Borders made of `=` describe all columns.
- **Header:** Separated by `=` underline.
- **Column Spans:** Indicated by `-` underline of joined columns.
- **Rules:** Each line of text starts a new row, except when there is a blank cell in the first column (continuation line).

```rst
=====  =====  =======
  A      B    A and B
=====  =====  =======
False  False  False
True   False  False
False  True   False
True   True   True
=====  =====  =======

=====  =====  ======
   Inputs     Output
------------  ------
  A      B    A or B
=====  =====  ======
```

To start a new row in a simple table without text in the first column, use an empty comment (`..`) or a backslash escape followed by a space (`\ `).
