# Style Guide

Based on the official Google Style Guide.

## Naming Conventions
| Element | Case Style | Example |
| :--- | :--- | :--- |
| **Message** | `PascalCase` | `SongServerRequest` |
| **Service** | `PascalCase` | `SpotifyService` |
| **Enum** | `PascalCase` | `Foo` |
| **Method** | `PascalCase` | `GetSong` |
| **Field** | `lower_snake_case` | `song_name` |
| **File Name** | `lower_snake_case` | `song_server_request.proto` |
| **Enum Value**| `UPPER_SNAKE_CASE` | `FOO_UNSPECIFIED` |

## Formatting
-   **Indent**: 2 spaces.
-   **Line Length**: Max 80 characters.
